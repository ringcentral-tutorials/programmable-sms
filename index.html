Environment Variables

The first file we'll look at is .env.sample. This contains all the environment variables we used in this tutorial. You need to follow its sample to create .env file which contains real RingCentral credentials.

Most of the environment variables are quite self-explanatory except the last one: RINGCENTRAL_RECEIVER. It means the number to receive SMS. You can set it to your own mobile phone number so that you can confirm the receipt of SMS.
